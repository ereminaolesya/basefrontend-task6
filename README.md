## Описание

Реализация  HTML+CSS проект адаптивного навигационного меню с использованием Flexbox.
Задача взята из учебного практического курса.

## Требования к реализации
Flexbox-позиционирование:
обязательно использовать display: flex для основной навигации
обязательно использовать flex-свойства для выравнивания элементов
Семантичность:
Использование тега <nav> для навигации
Правильная структура списка (<ul>, <li>)
Использование тега <a> для лого сайта и навигационных ссылок
